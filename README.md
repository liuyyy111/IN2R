
# Intra-Modal Neighbors Never Lie: Rectifying Inter-Modal Noisy Correspondence via Graph-Based Intra-Modal Reasoning

The official codes for our paper "Intra-Modal Neighbors Never Lie: Rectifying Inter-Modal Noisy Correspondence via Graph-Based Intra-Modal Reasoning", which is accepted by the International Conference on Machine Learning, ICML 2026.
We referred to the implementations of [NCR](https://github.com/XLearning-SCU/2021-NeurIPS-NCR), [GPO](https://github.com/woodfrog/vse_infty), and [ConVSE](https://github.com/liuyyy111/ConVSE) to build up the repository.
